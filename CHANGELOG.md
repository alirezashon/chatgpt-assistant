# Changelog

All notable changes to ChatGPT Workspace are tracked here.

## 0.1.0 - 2026-07-10

### Added

- Local-first Chrome extension foundation for ChatGPT workspace organization.
- Sidebar folder management, current conversation assignment, favorites, and local search.
- Account-free Free Local plan and Pro feature catalog.
- Markdown export for selected conversation metadata.
- Local JSON backup export and import from the options page.
- First-run onboarding, empty-state guidance, and ChatGPT DOM detection error messaging.
- Release notes, privacy promises, and keyboard shortcut documentation in settings.
- Vitest unit coverage and Playwright extension smoke tests.
- Bundle size reporting through `npm run bundle:size`.

### Notes

- No account, billing, backend, cloud sync, or external AI provider is enabled yet.
- Paid features are represented as gated product definitions only.
