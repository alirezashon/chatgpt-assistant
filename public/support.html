ChatGPT Workspace Support

Local beta support guide

Before reporting an issue

  • Load the latest build from the dist folder in Chrome extension developer mode.
  • Reload the extension and any open ChatGPT tabs.
  • Open the options page and confirm local storage is available.
  • Export a local backup before testing destructive workspace changes.

Useful details to include

  • Extension version and Chrome version.
  • The ChatGPT URL where the issue happened.
  • Expected behavior, actual behavior, and whether the sidebar showed an error state.
  • Steps that reliably reproduce the issue.

Current support status

The current beta includes an email draft link from the options page. Add the final public support address and Chrome Web Store support URL before public release.
